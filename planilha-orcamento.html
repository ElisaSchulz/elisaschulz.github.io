Como usar e exportar para Excel

1
Selecione o mês que deseja planejar no menu acima. Cada mês é salvo separadamente no seu navegador.
2
Preencha seus rendimentos (salário, freelas, etc.) e suas despesas por categoria. Você pode editar os nomes dos itens clicando sobre eles e adicionar novas linhas.
3
A coluna "Previsto" é o que você planeja gastar. A coluna "Real" é o que efetivamente gastou. Compare os dois para entender seus desvios.
4
Clique em "Exportar CSV" para baixar o orçamento. Para abrir no Excel: clique com o botão direito no arquivo → Abrir com → Excel → na importação escolha ponto-e-vírgula como separador.
5
Os dados ficam salvos no seu navegador (localStorage). Se limpar o histórico do navegador, os dados serão apagados. Exporte periodicamente para não perder.
